# Frequent Subgraph Mining under Differential Privacy in a Single Graph

This repository provides implementations of algorithms for mining frequent subgraphs under **differential privacy** in a **single graph**. The framework supports two support measures:

- **MVC** (Minimum Vertex Cover)
- **MIS** (Maximum Independent Set)

## Requirements

- **Java 8** or later.
- All required libraries and project dependencies should be properly configured.

## Getting Started

### Step 1: Open the Project

Open the project in **IntelliJ IDEA**. The IDE will automatically detect and compile the Java source files.

### Step 2: Run the Exact Mining Algorithm

Execute `MainTest.java`, which serves as the entry point for the exact frequent subgraph mining algorithm.

In IntelliJ IDEA:

1. Right-click `MainTest.java`.
2. Select **Run 'MainTest'**.

Alternatively, open `MainTest.java` and click the **Run** button in the toolbar.

### Step 3: Configure the Parameters

Before running the program, you can modify the following parameters:

- **`minSupport`**  
  The minimum support threshold for frequent subgraph mining.

- **`supType`**  
  The support measure used during mining. Available options are:
  - `"MVC"`
  - `"MIS"`

- **`isNaive`**  
  A boolean flag indicating whether to use the naive differentially private algorithm.
  - `true`: Use the naive algorithm.
  - `false`: Use the proposed algorithm.

### Step 4: Run the Differentially Private Algorithm

Execute `MainTestDP.java` to run the differentially private frequent subgraph mining algorithm.

### Step 5: Evaluate the Results

Execute `Performance.java` to evaluate the mining results.

This program compares the output files generated by:

- `MainTest.java` (`output.txt`)
- `MainTestDP.java` (`outputDP.txt`)

The following evaluation metrics are reported:

- Precision
- Recall
- F-score
- Relative Error
- Absolute Error

## Example Configuration

The following example runs the proposed algorithm on the **Deezer** dataset with a minimum support threshold of **20**, using the **MVC** support measure.

```java
String input = "Data/deezer.lg";
int minSupport = 20;
String supType = "MVC";
boolean isNaive = false;
```
